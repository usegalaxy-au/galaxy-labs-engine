Requesting installation of a tool

If you need to request installation of a tool on a Galaxy server, the process varies depending on the server's policies and procedures. For the three main Galaxy servers, you can make a pull-request to the following GitHub repositories for automated tool installation.

(Don't worry, instructions are included!)

usegalaxy.org https://github.com/galaxyproject/usegalaxy-tools
usegalaxy.eu https://github.com/usegalaxy-eu/usegalaxy-eu-tools
usegalaxy.org.au https://github.com/usegalaxy-au/usegalaxy-au-tools
